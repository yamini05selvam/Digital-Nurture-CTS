When you download Bootstrap manually, the extracted folder usually looks like this:

```text id="cb2zy0"
bootstrap/
│
├── css/
├── js/
└── icons/   (optional if Bootstrap Icons are downloaded separately)
```

# 1. CSS Folder

Example:

```text id="7g7n52"
css/
│
├── bootstrap.css
├── bootstrap.min.css
├── bootstrap.rtl.css
└── bootstrap.rtl.min.css
```

## Purpose

The CSS folder contains all Bootstrap styling files.

These files provide:

* Grid system
* Buttons
* Forms
* Cards
* Navbar
* Colors
* Spacing utilities
* Responsive design

---

## Important Files

### `bootstrap.css`

Readable normal version.

* Used for learning/debugging
* Large file size
* Easy to edit

### `bootstrap.min.css`

Minified production version.

* Removes spaces/comments
* Faster loading
* Mostly used in real projects

### `bootstrap.rtl.css`

RTL = Right-To-Left support.

Used for languages like:

* Arabic
* Urdu
* Hebrew

---

# 2. JS Folder

Example:

```text id="u0g7qj"
js/
│
├── bootstrap.js
├── bootstrap.bundle.js
├── bootstrap.min.js
└── bootstrap.bundle.min.js
```

## Purpose

Contains Bootstrap JavaScript functionality.

Used for interactive components like:

* Modals
* Dropdowns
* Carousel
* Collapse
* Toasts
* Tooltips
* Navbar toggles

---

## Important Files

### `bootstrap.js`

Normal readable JavaScript file.

### `bootstrap.min.js`

Compressed/minified version.

### `bootstrap.bundle.js`

Includes:

* Bootstrap JS
* Popper.js

Popper is needed for:

* Tooltips
* Dropdown positioning

### `bootstrap.bundle.min.js`

Minified production version of bundle.

Most commonly used file.

---

# 3. Icons Folder

Bootstrap icons are usually downloaded separately from:

[Bootstrap Icons](https://icons.getbootstrap.com/?utm_source=chatgpt.com)

Example structure:

```text id="k8q0xr"
icons/
│
├── font/
├── icons/
└── bootstrap-icons.css
```

## Purpose

Provides ready-made icons.

Examples:

* Search icon
* Person icon
* Heart icon
* Trash icon
* Menu icon

---

## Important Parts

### `bootstrap-icons.css`

Main stylesheet for icons.

### `icons/`

Contains SVG icon files.

### `font/`

Contains icon font files.

---

# Example Usage of Bootstrap Icons

```html id="y8r6im"
<link rel="stylesheet" href="icons/bootstrap-icons.css">

<i class="bi bi-alarm"></i>
<i class="bi bi-heart-fill"></i>
```

---

# Summary

| Folder | Purpose                          |
| ------ | -------------------------------- |
| css    | Styling and responsive layout    |
| js     | Interactive Bootstrap components |
| icons  | Ready-made SVG/font icons        |
